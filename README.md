# Homebrew Tap

This repository provides Homebrew formulae for unofficial software packages.

## Tap installation

    brew tap maurymarkowitz/tap

## Install OpenNEC/onec

    brew install maurymarkowitz/tap/onec

## Install RetroJOSS

    brew install maurymarkowitz/tap/retrojoss

## Install RetroFOCAL

    brew install maurymarkowitz/tap/retrofocal

## Install RetroBASIC

    brew install maurymarkowitz/tap/retrobasic

## Notes

- `retrofocal` is available at version 1.1.4.
- `retrobasic` is available at version 2.1.2.
- Both require `flex` and `bison` for build-from-source installs.
- If packages are not prebuilt, use `brew install --build-from-source`.
